Verify a kry receipt

Checks a kry attestation entirely in your browser — nothing is uploaded, there is no server to trust. The verifier is a single file, verify.mjs (view its source — it is the same implementation that passes a 1,000,000-case differential fuzz against the reference, per KRY-SPEC v1.0).

A VALID verdict means the receipt's hash chain, magnitude arithmetic, tier schema and self-hash all re-derive correctly — i.e. no field was altered. It does not attest that the savings were real; that is the receipt's veracity_floor and evidence tier to disclose, not this checker to certify.
